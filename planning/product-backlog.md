# User Stories

## Patient
- As a patient, I want to create an account with my personal information, so that I can start booking appointments.
- As a patient, I want to book an appointment at the hospital through digital means, so that I do not have to wait in long queues.
- As a patient, I want to received a detailed booking confirmation, so that I have proof and all necessary information.
- As a patient, I want to see all my past and upcoming appointments, so that I can track my medical visits.
- As a patient, I want to cancel my appointment if I would not be able to attend, so that the slot becomes unoccupied.

## Doctor
- As a doctor, I want to view my daily appointment schedule, so that I know which patients I am seeing today.
- As a doctor, I want to call the next patient in the queue, so that they know it is their turn.
- As a doctor, I want to document consultation findings, so that there is a proper medical record.
- As a doctor, I want to digitally prescribe medications, so that patients can directly collect from the pharmacy.
- As a doctor, I want to issue medical certificates to patients, so that they can submit it to their employer or school.
- As a doctor, I want to mark consultation as complete, so that I can move onto the next patient.

## Administrator
- As an administrator, I want to set the hospital's operating hours, so that the patient can only book during open hours.
- As an administrator, I want to send announcements to all patients, so that everyone is informed of important updates.
- As an administrator, I want to see all appointments across all doctors, so that I can monitor the system operations.
