# User Stories

## Patients
- As a patient, I want to create an account with my personal information, so that I can start booking appointments.
- As a patient, I want to book an appointment at the hospital through digital means, so that I do not have to wait in long queues.
- As a patient, I want to received a detailed booking confirmation, so that I have proof and all necessary information.
- As a patient, I want to see all my past and upcoming appointments, so that I can track my medical visits.
- As a patient, I want to change my appointment to a different date and time, so that I can be flexible with my schedule.
- As a patient, I want to cancel my appointment if I would not be able to attend, so that the slot becomes unoccupied.
- As a patient, I want to receive reminders on my appointment, so that I do not forget about it.
- As a patient, I want to see the current queue status at the hospital, so that I can plan my arrival time.
- As a patient, I want to want to store medical history and conditions, so that the doctors have a complete medical record of me.
- As a patient, I want to store a list of emergency contacts, so that the doctors can reach out to someone if needed.

## Medical Staff
- As a medical staff, I want to view my daily appointment schedule, so that I know which patients I am seeing today.
- As a medical staff, I want to call the next patient in the queue, so that they know it is their turn.
- As a medical staff, I want to document consultation findings, so that there is a proper medical record.
- As a medical staff, I want to digitally prescribe medications, so that patients can directly collect from the pharmacy.
- As a medical staff, I want to issue medical certificates to patients, so that they can submit it to their employer or school.
- As a medical staff, I want to mark consultation as complete, so that I can move onto the next patient.
- As a medical staff, I want to view all patients currently waiting, so that I can manage patient flow.
- As a medical staff, I want to mark patients who did not attend as absent, so that the next patient can be called.

## Administrators
- As an administrators, I want to set the hospital's operating hours, so that the patient can only book during open hours.
- As an administrators, I want to send announcements to all patients, so that everyone is informed of important updates.
- As an administrators, I want to see all appointments across all doctors, so that I can monitor the system operations.
- As an administrators, I want to create appointments on behalf of patients, so that I can help those who call or walk-in.
- As an administrators, I want to cancel appointments if needed, so that I can handle emergencies or doctor unavailability.
